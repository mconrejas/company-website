Create your website using Elementor Page Builder for free. Build your header, footer, posts, pages or archive blog page with Elementor.

== Description ==

Theme Builder For Elementor is as powerful free WordPress plugin that allows you to create almost entire website with Elementor.
- Header 
- Footer 
- Post
- Page
- Sidebar
- Archive (Blog) page
- 404 page

[Theme Builder Documentation](https://blocks-wp.com/blog/docs/theme-builder-for-elementor-documentation/ "Theme Builder For Elementor Documentation")

= Comes With Inbuilt Widgets =

The Theme Builder For Elementor comes with inbuilt widgets that help create site layouts.

<ul>
    <li><strong>Post title</strong> - Post title will be automatically inherited from the post/page.</li>
    <li><strong>Content</strong> - Content (or excerpt) will be automatically inherited from the post/page.</li>
    <li><strong>Post date</strong> - Date will be automatically inherited from the post/page.</li>
    <li><strong>Featured image</strong> - Featured image will be automatically inherited from the post/page.</li>
    <li><strong>Read more button</strong> - Link of the button will be automatically inherited from the post/page - only for archive/blog.</li>
    <li><strong>Post author</strong> - Author will be automatically inherited from the post/page.</li>
    <li><strong>Post categories</strong> - Categories will be automatically inherited from the post (pages has no categories).</li>
    <li><strong>Post tags</strong> - Tags will be automatically inherited from the post (pages has no categories).</li>
    <li><strong>Post comments</strong> - Comments count will be automatically inherited from the post/page.</li>
    <li><strong>Sidebar & widgets</strong> - Style sidebar/widgets layouts and put them into archive/blog or post/page.</li>
</ul>

= One click demo layout import =

The Theme Builder For Elementor comes with pre-designed headers, footers, posts and pages, archives and 404 pages. Import them with single click and use them for your new website development. 

= Supported themes =

Elementor theme builder is tested with several themes. But not all themes can be overridden with this plugin. We have developed very simple and lightweight (less than 8kb) theme that can be overridden with this plugin and it is 100% compatible with it. For the best performance, it is recommended to use our theme. To install it go to your WP Dashboard - Theme Builder - Settings.

== Installation ==

1. Go to 'Plugins -> Add New' and search for Theme Builder For Elementor.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to 'Theme Builder' to build sections.
4. After the layout is ready assign it go to 'Theme Builder -> Settings' and assign the template

== Frequently Asked Questions ==

= Works with all themes? =

Just now is the plugin tested with few themes. The list of supported themes will be soon. For the best performance, it is recommended to use our theme. To install it go to your WP Dashboard - Theme Builder - Settings.

= Will it work without Elementor =

No. This is an Elementor addon. It will not work without free Elementor plugin.

= Is Elementor PRO required? =

No. The plugin is built for free Elementor plugin. Elementor PRO is not required.

= What is the difference between this plugin and Elementor PRO? =

This is free plugin that will help you develop your sites. The Elementor PRO theme builder is a paid feature of Elementor PRO. Elementor PRO theme builder is not compatible with this plugin. Use only one theme builder, do not combine them.
